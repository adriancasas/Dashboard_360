# **App Name**: Agent Dashboard

## Core Features:

- Agent Status Overview: Display the current status (active/inactive) of each agent.
- Agent Credit Balance: Show the available credits for each agent (e.g., 500 credits left / Agent Y (0 credits left - recharge)).
- Marketplace Integration: Connect to the user's agent marketplace to automatically sync agent data and contract information.
- Alerting and Notifications: Set up custom alerts and notifications based on agent status or performance metrics (e.g., notify the user when an agent has his task done).
- User Profile: Display the user profile information.
- Project Organization: Organize projects by folders.
- Agent Assignment to Projects: Assign agents to specific projects.
- Team Sharing: Share projects with teammates.
- Agent Addition to Projects: Add agents to existing projects.
- Total Available Credits: Display the total available credits for activating new agents.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust, stability, and professionalism.
- Background color: Light Gray (#F5F5F5) to ensure a clean, modern, and unobtrusive backdrop that enhances data readability.
- Accent color: Teal (#38B2AC) for interactive elements and key performance indicators to draw user attention and indicate actionable items.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look suitable for both headlines and body text
- Use clear, consistent icons to represent agent statuses, metrics, and actions.
- Implement a dashboard layout with clear sections for each agent and their key metrics. Use card-based components to organize information and improve readability.
- Incorporate subtle transitions and animations to provide feedback and enhance the user experience, without being distracting.
